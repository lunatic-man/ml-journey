# Concepts
This is just a rough collection of all concepts that I have studied so far. The explanation is in my own words and will continuously change and evolve.

1. venv - isolated package shelf per project 
2. variables - stores reference to data sitting in memory, can be manipulated
3. constants - stores reference to data sitting in memory, cannot be changed
4. functions - set of instructions that return some object (Always returns an object, empty ones return None object)
5. return - used to return a particular value from a function
6. docstrings - documentation that is added to a function to explain what it does and usually includes how many parameters it takes and what does it return. (enclosed between """ """).
7. boolean operators - `and`, `or` and `not` are the three operators that can be used to manipulate boolean values
8. The output of a function can be used as the input for another function, often times in a single line. Use this property to make a clean code.
